# SimCit

A retro isometric sim city clone project.

## Project Resources

See [PROJECTS.md](PROJECTS.md) for a comprehensive list of open-source projects, game engines, libraries, and resources that can be used to build this game.

## Overview

This repository will contain a retro-style isometric city-building simulation game inspired by classics like SimCity and SimCity 2000. The project aims to recreate the nostalgic gameplay and visual style of these beloved titles using modern web technologies.

## Getting Started

To explore potential frameworks and engines for this project, see the curated list in [PROJECTS.md](PROJECTS.md), which includes:
- Complete open source city-building games
- Game engines and frameworks
- JavaScript/HTML5 libraries
- Rendering and asset libraries
- Classic SimCity ports
